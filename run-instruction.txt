Para correr el programa, luego de instalar los modulos (\api-rest-aenima\myapp> npm i) se debe correr la base de datos que vendra en un archivo .zip,
la ruta de la base de datos viene por defecto en el puerto 27017, por si es necesario cambiar el puerto.
El programa incluye nodemon, asique se recomienda ejecutar el servidor usando (\api-rest-aenima\myapp> nodemon start)
